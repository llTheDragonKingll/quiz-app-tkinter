# 🧠 Quiz App (Python Tkinter)

A simple quiz application built using Python and Tkinter that displays questions one by one and keeps track of the score.

---

## 🚀 Features

* ❓ Displays quiz questions in a clean GUI
* ✔️ True / False answer system
* 📊 Real-time score tracking
* 🎨 Image-based buttons for answers
* ⚡ Instant feedback after each response

---

## 🛠️ Tech Stack

* Python
* Tkinter
* Object-Oriented Programming

---
